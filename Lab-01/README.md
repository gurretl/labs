# Prerequisites
* Install Oracle Virtual Box  
* Install Vagrant
# How to launch the Lab
Download the code  
Go to this directory and run :  
`vagrant up`  
Then, run the following command to get ssh config to reach your VMs :  
`vagrant ssh-config`
