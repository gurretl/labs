# How to run the lab
`git clone https://github.com/gurretl/labs.git`  
`cd labs/Lab-21`  
`./build-images.sh`  
Edit run.sh script to specify the number of hosts you need for your Lab (NOF_HOSTS)  
`./run.sh`
